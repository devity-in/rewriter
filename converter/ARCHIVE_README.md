# Archive

This directory contains archived converter scripts and documentation from earlier development.

## Archived Files

- `convert_phi3*.py` - Old Phi-3 specific converters
- `convert_gemma.py` - Old Gemma-specific converter (superseded by `convert_model.py`)
- Old documentation files (QUICKSTART.md, STATUS.md, etc.)

## Current Converter

Use `convert_model.py` for all model conversions. It supports:
- Gemma models (GEMMA3_300M, GEMMA3_1B, etc.)
- Other Mediapipe-compatible models

See `README.md` for usage instructions.
